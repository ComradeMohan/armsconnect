# ARMSConnect

A modern, highly-animated, responsive student portal interface built with Next.js and React.

## Features
- Secure Login with auto-sync and local storage persistence
- Dashboard with real-time academic stats
- Beautiful dark-mode Glassmorphism UI
- Fully responsive (Mobile & Desktop)
- Animated 404 page

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
